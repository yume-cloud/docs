## **Что это за раздел**

Раздел **«Клиенты»** — это единая база всех ваших клиентов или водителей.

Здесь хранится:

- контактная информация
- история аренд
- платежи
- активность
- рейтинг клиента

Этот раздел используется каждый день —\
все операции в системе начинаются с клиента.

## Как открыть раздел

В левом меню нажмите:

→ **«Клиенты»**

После этого откроется список всех клиентов.

## Как устроен экран

### Поиск

В верхней части есть строка поиска.

Вы можете найти клиента по:

- имени
- номеру телефона
- названию компании

Просто начните вводить — система сразу покажет результат.

## Фильтры

Рядом с поиском находятся фильтры.

### Тип клиента

Позволяет выбрать:

- физическое лицо
- юридическое лицо

### Канал привлечения

Показывает, откуда пришёл клиент:

- Instagram
- WhatsApp
- рекомендации\
  и другие источники

Это важно для анализа маркетинга.

### Рейтинг клиента

Позволяет отфильтровать клиентов по надёжности:

- порядочный
- постоянный
- проблемный
- мошенник

## Статистика

Ниже отображается сводная информация по базе:

- общее количество клиентов
- процент повторных клиентов
- клиенты с просрочками
- средний чек

Это помогает быстро оценить состояние бизнеса.

## Таблица клиентов

Основная часть экрана — это таблица.

В каждой строке отображается:

- имя или название компании
- тип клиента
- сумма аренд
- количество аренд
- баланс
- канал привлечения
- дата последней аренды
- рейтинг

## Рейтинг клиента

Система позволяет отмечать клиентов:

- надёжный
- VIP / Gold
- проблемный
- мошенник

Это помогает быстро принимать решения\
при создании новой аренды.

## Бонусы

Если в системе используется бонусная программа,\
в таблице отображается бонусный баланс клиента.

## Карточка клиента

Чтобы открыть подробную информацию:

→ нажмите на имя клиента

## Что вы увидите

В карточке клиента есть:

- история всех аренд
- платежи
- задолженности
- баланс
- заметки и взаимодействия

Это полный профиль клиента.

## Зачем это нужно

Вы можете:

- проверить историю клиента перед новой сделкой
- увидеть долги
- понять, насколько он надёжный

# Как добавить клиента

В правом верхнем углу нажмите:

→ **«Добавить»**

## Обязательные поля

- ФИО
- номер телефона

## Дополнительно (рекомендуется)

- ИИН
- паспортные данные
- дата рождения
- фото
- email

Чем больше данных — тем лучше контроль.

## Дополнительные настройки

При создании можно указать:

### Канал привлечения

Чтобы понимать, откуда пришёл клиент

### Скидка

Постоянная скидка, которая будет применяться автоматически

### Рейтинг

Оценка надёжности клиента

## После создания

Клиент:

- сохраняется в базе
- становится доступен при создании аренды
- участвует в аналитике

## Экспорт клиентов

Нажмите:

→ **«Экспорт таблицы»**

Вы можете:

- скачать список клиентов
- использовать данные для отчётов
- анализировать базу

## Как используется в работе

Каждая аренда связана с клиентом.

Это значит:

- не нужно создавать клиента каждый раз
- вся история сохраняется автоматически

## Важно понять

Клиенты — это основа системы.

Если база заполнена правильно:

- быстрее создаются аренды
- меньше ошибок
- проще контролировать бизнес

## Если коротко

1. Добавьте клиента
2. Заполните данные
3. Используйте его в арендах
4. Следите за историей и оплатами
