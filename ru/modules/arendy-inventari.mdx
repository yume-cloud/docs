## **Что это за раздел ?**

Раздел используется для работы с арендой оборудования, техники и других товаров.

Вы управляете: выдачей и возвратом инвентаря, сроками аренды, состоянием товаров, оплатами и залогами. Это основной сценарий для проката.

### Как устроен экран

Вкладки (статусы)

В верхней части находятся вкладки: Все, Запросы, Забронировано, В аренде, Завершено, Просрочено, Должники, Отменённые

Каждая вкладка показывает аренды в определённом состоянии.

### Поиск

В строке поиска можно ввести: имя клиента, номер телефона, название товара, артикул.

Система сразу покажет нужную аренду.

### Фильтр по датам

Есть мини-календарь. Вы можете: выбрать текущий месяц, посмотреть прошлый период, задать свой диапазон дат.

### Пункт проката

Если у вас несколько точек: → выберите нужную → увидите аренды только по ней.

### Фильтр оплаты

Есть три статуса:

Ожидает оплату — оплат не было

Частично оплачено — внесена часть

Оплачено — полностью закрыта

### Вид отображения

Можно переключаться: карточки, таблица.

Таблица удобнее при большом количестве данных.

### Статистика и экспорт

Доступно: общая сумма аренд, сумма оплат, задолженность, экспорт в Excel.

### Статусы аренды (очень важно)

**Запросы**

Черновик аренды. Вы создали аренду, но ещё не подтвердили.

**Забронировано**

Аренда запланирована на будущее. Клиент ещё не забрал товар.

**В аренде**

Товар уже у клиента.

**Завершено**

Товар возвращён, оплата закрыта. Аренда больше не требует действий.

**Просрочено**

Клиент не вернул товар вовремя. Система ставит статус автоматически.

**Должники**

Товар возвращён, но оплата не закрыта полностью.

### **Как устроена аренда инвентаря ?**

Каждая аренда включает: клиента, период аренды, список товаров, финансовые операции, документы и заметки.

Все элементы связаны между собой внутри одной карточки.

### **Как создать аренду ?**

1. **Нажмите “Новая аренда”.** Кнопка находится в левом меню.
2. **Выберите клиента**. Если клиент есть: → найдите через поиск Если нет: → нажмите “Новый клиент” **Создание клиента** Заполните:
   - ФИО
   - телефон Дополнительно (рекомендуется):
   - ИИН
   - паспорт
   - дата рождения
   - фото Это поможет лучше контролировать клиентов. Дополнительно можно указать:
   - канал привлечения (Instagram, WhatsApp и т.д.)
   - постоянную скидку
   - рейтинг клиента Это используется для аналитики и контроля.
3. **Задайте период**: дата и время начала, дата и время окончания. Система рассчитывает длительность автоматически. Важно: если есть \+1 минута → считается как \+1 день
4. **Добавьте инвентарь.** В блоке Продукты / Комплекты: нажмите “Добавить”, выберите нужные позиции. Важно: товары должны быть заранее добавлены в Каталог. Можно:
   - включить “Только свободные”
   - фильтровать по категории Цена:
   - подтягивается автоматически
   - или вводится вручную
5. **Сохраните аренду.** Нажмите “Сохранить изменения”. Аренда перейдёт в статус → Забронировано

### **Работа с арендой**

**Оплата**

Нажмите “Принять оплату”. Введите:

- сумму
- способ оплаты

Система создаст чек.

Можно:

- принимать частями
- закрывать позже

**Залог**

Можно добавить:

- сумму
- способ оплаты

Пока не возвращён — отображается в системе.

При возврате → нажмите “Вернуть”

Используется для защиты инвентаря: указывается сумма, возвращается после завершения аренды

**Штрафы**

Можно:

- добавить вручную
- включить автоштраф (по часам)

**Заметки и файлы**

Используются для: комментариев, документов, внутренних записей

### **Завершение аренды**

**Шаг 1. Принять товар**

Нажмите “Принять”. Выберите состояние: исправен / сломан

Если сломан → товар временно недоступен

**Шаг 2. Завершить аренду**

Нажмите “Завершить”

Что произойдёт

Если: всё оплачено → статус “Завершено”, есть долг → статус “Должники”

**Закрытие долга**

Когда клиент оплатит → нажмите “Принять оплату”

После этого аренда полностью закрывается.

### Важно понять

Всё крутится вокруг аренды:

Клиент, товар и деньги связаны. Ничего не нужно вводить дважды.

**Если коротко**

Чтобы работать: добавь клиента, добавь товар, создай аренду, прими оплату, заверши

**Когда использовать ?**

Используйте этот раздел, если вы:

сдаёте оборудование / сдаёте технику / работаете с физическим инвентарём
