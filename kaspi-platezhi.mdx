Раздел «Kaspi Платежи» предназначен для автоматизации приема платежей от клиентов и водителей. После подключения система берет на себя учет оплат, распределение средств и контроль задолженностей.

**Подключение интеграции**\
Для начала работы необходимо нажать кнопку «Подключить». После подключения система начинает автоматически обрабатывать все платежи через Kaspi.

**Основная логика работы**\
Система связывает платежи с конкретными пользователями через ID. При оплате:

- водитель или клиент вводит свой ID
- система автоматически определяет, кому принадлежит платеж
- деньги сразу отображаются в системе

Это исключает ручную проверку переводов.

**Автоматическое зачисление платежей**\
Все поступления фиксируются в реальном времени.\
Менеджеру не нужно:

- проверять переводы вручную
- сверять платежи
- уточнять, кто оплатил

Система делает это автоматически.

**Автосписания**\
Можно настроить автоматические списания:

- аренда
- штрафы
- дополнительные услуги

Списания происходят по расписанию без участия менеджера.

**Комиссия**\
При использовании интеграции применяется сниженная комиссия (например, 2.5%), которая ниже стандартной комиссии при прямых переводах через Kaspi.

**Прозрачность операций**\
Все операции фиксируются в системе:

- кто оплатил
- когда оплатил
- сумма платежа
- итог с учетом комиссии

Это позволяет полностью контролировать денежные потоки.

**Журнал транзакций**\
В разделе доступен полный журнал:

- пополнения
- списания
- остатки

Все финансовые операции находятся в одном месте.

**Контроль балансов**\
Система позволяет отслеживать:

- баланс каждого пользователя
- задолженности
- переплаты

Это помогает избегать финансовых ошибок.

**Проблемы без интеграции**\
Без подключения:

- нет контроля оплат
- менеджеры тратят время на ручные проверки
- нет автосписаний
